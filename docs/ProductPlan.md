# TimelineAnimations Product Plan

## Product Vision
- Deliver a desktop-grade Avalonia animation editor for keyframe-based motion design.
- Make the editor feel immediate and visual: compose objects on a canvas, edit them directly, and tune motion from a dedicated timeline surface.

## Core User Experience
- WYSIWYG stage with live playback.
- Layer-based composition for rectangles, ellipses, and text elements.
- Keyframe timeline for position, size, rotation, and opacity.
- Inspector panel for precise numeric editing and color/text updates.
- Drag and drop palette for adding new elements to the stage.

## MVP Scope
- Desktop shell with polished layout and custom styling.
- Sample composition loaded on startup.
- Canvas selection, move, resize, and live preview.
- Timeline scrubbing, keyframe selection, keyframe drag, and add/remove keyframes.
- Playback controls with looping.
- Save and open timeline documents as JSON.

## Quality Targets
- Fast feedback with no blocking UI during playback.
- Strong visual identity with layered surfaces, contrast, and color accents.
- Test coverage for interpolation, keyframe editing, and serialization.
